# AES
Open source implementation of the Advanced Encryption Standard (AES) algorithm in the Rust programming language.
The project is current still in the pre-alpha phase.

## Roadmap
Upcoming features:
- Key Expansion
- Convert Text input into the states
- decryption function
- full unit tests of all components
- cargo deployment
